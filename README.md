 📰 Current Affairs – News Aggregator Web App

A modern full-stack News Aggregator Web Application that fetches and displays real-time news from multiple categories using NewsAPI. The application provides a clean, responsive interface with features like breaking news, search, dark mode, authentication, and pagination.

 🚀 Features

 📢 Breaking News Ticker – Live scrolling headlines displayed at the top.
 📰 7 News Categories

   🇮🇳 India
   🌍 Top News
   💻 Technology
   🏏 Sports
   🎬 Entertainment
   💼 Business
   🧪 Science & Health
 🎞️ Auto Slider – Top 5 featured news articles displayed with automatic sliding, navigation arrows, and dot indicators.
 🔍 Search Functionality – Search for any news topic with trending quick tags.
 🌙 Dark Mode – Toggle between light and dark themes with user preference stored in Local Storage.
 📄 Load More – Pagination support for loading additional news articles.
 🔐 Authentication – Login and Signup with form validation and session management.
 📱 Responsive Design – Optimized for desktop, tablet, and mobile devices.

 🛠️ Tech Stack

| Layer          | Technology                                  |
| -------------- | ------------------------------------------- |
| Frontend       | HTML5, CSS3, JavaScript (ES6+)              |
| API            | NewsAPI.org                                 |
| Authentication | JavaScript (Client-side Session Management) |

 ⚙️ Installation

1. Clone the repository:

```bash
https://github.com/ChandanSoni1611/current-affairs-news-website
```

2. Navigate to the project directory:

```bash
cd CurrentAffairs
```

3. Get a free API key from NewsAPI.org.

4. Add your API key to the JavaScript configuration file:

```javascript
const API_KEY = "YOUR_NEWSAPI_KEY";
```

5. Open `index.html` using Live Server or deploy the project on Render.
